# kotlinx-jdbc
A simple JDBC wrapper in Kotlin

# Introduction to kotlinx-jdbc

**kotlinx-jdbc** is a simple JDBC wrapper in Kotlin. It provides a simple and easy-to-use API for working with JDBC. 
It is designed to be lightweight and easy to use, while still providing all the features you need to work with JDBC.

**kotlinx-jdbc** is not an ORM (Object-Relational Mapping) library. 
It's a simple wrapper around JDBC that provides a more Kotlin-friendly API. The library aims to make operations with JDBC
explicit wwith no hidden magic.
